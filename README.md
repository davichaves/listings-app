## Installation and Setup Instructions

#### Example:

Clone down this repository. You will need `node`, and `npm`, and `yarn` installed globally on your machine.

Installation:

`yarn`

To Start Server:

`yarn start`

To Visit App:

`localhost:3000`

To build the app:

`yarn build`
